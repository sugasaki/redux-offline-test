


## redux-offline

redux-offlineのExampleの動作確認用

[redux-offline/redux-offline: Build Offline-First Apps for Web and React Native](https://github.com/redux-offline/redux-offline)




## how to use

1. サーバーを起動
2. クライアントを起動



## サーバー起動

```
cd server
npm i
npm start
```


## クライアント起動

```
cd client
npm i
npm start
```


access to

http://localhost:3000/
